How Rounds Work

  • In the experiment, you will work with Points. At the end of the experiment, Points will be converted to Dollars. Todayâ€™s exchange rate is {{ exRate }} Points = $1
  • The experiment is made up of {{ Rounds }} Rounds.
  • Each round will last for a random number of Periods:
    • At the end of each period the computer will generate a random number between 1 and {{ dieN }} ({1, 2,..., {{ dieN }}}) with each number being equally likely.
    • If the computer generates a number less than {{ thres }}, then the Round continues for at least one more period ({{ contProb }}% Probability).
    • If the computer generates a number equal to {{ thres }} or greater, then the Round ends ({{ endProb }}% Probability).
  • In each period of a round you will participate in two games
    Red Game
    and
    Blue Game
    simultaneously. You will participate in each game with a different participant.
  • At the start of each round, you will be paired with two other randomly chosen participants, one for
    Red Game
    and one for
    Blue Game
    . We will refer to the participant you will be paired with for
    Red Game
    as Other Red and the participant you will be paired with for
    Blue Game
    as Other Blue.
  • You will then participate in
    Red Game
    and
    Blue Game
    simultaneously for a random number of periods with Other Red and Other Blue respectively.
  • What happens when you can not be paired with other participants in a round?
    • You will be informed if you could not be paired.
    • You will be allotted {{ expPoints_Red }} points for
      Red Game
      for each period of that round.
    • You will be allotted {{ expPoints_Blue }} points for
      Blue Game
      for each period of that round.
